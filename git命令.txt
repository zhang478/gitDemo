//全局配置
git config --global user.name "your name"
git config --global user.email "your email"

//push和pull不在输入密码
git config --global credential.helper store


初始化本地仓库：
git init

检出克隆仓库:
本地仓库：git clone /path/to/repository
远程仓库：git clone username@host:/path/to/repository

关联远程仓库：
git remote add origin https://github.com/zhang478/gitDemo.git

更新pull远程仓库代码：
git pull origin master

添加：
git add <filename>
git add *

提交
git commit -m "代码提交信息"

推送改动，提交到远端仓库（可以把 master 换成你想要推送的任何分支）：
git push origin master   

创建分支并切换（分支需推送到远程仓库，不然他人不可见）：
git checkout -b feature
推送分支：
git push origin feature

查看分支：
git branch 
git branch -a

切换回主分支：
git checkout master

合并分支：
git merge feature

合并分支解决冲突后，标记合并成功：
git add filename

删除本地分支：
git branch -d feature

删除远程仓库分支：
git push origin --delete feature
